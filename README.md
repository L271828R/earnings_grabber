# Description

Would you like to have the following data from your favorite stocks saved to a database:

*  earnings dates 
*  market cap  
*  street address
*  business sector
*  industry
*  number of employees 

This program scrapes yahoo finance for just that information based on a list of tickers.

# Dependencies

* Python3
* selenium
* MongoClient

* Chrome Driver
